## 核心模型

| 模型 | 作用 |
|------|------|
| BERT (bert_query_classifier) | 查询分类：通用知识 / 专业咨询 |
| BGE-M3 | 文本向量化（稠密+稀疏） |
| BGE-Reranker-Large | 检索结果精排 |
| Qwen-Plus | 最终答案生成 |

## 目录结构

```
integrated_qa_system/
├── base/                          # 基础配置
│   ├── config.py                  # 配置加载
│   └── logger.py                  # 日志
├── rag_qa/                        # RAG 核心
│   ├── core/
│   │   ├── rag_system.py          # RAG 主流程

│   │   ├── document_processor.py  # 文档切分
│   │   ├── query_classifier.py    # 查询分类器
│   │   ├── strategy_selector.py   # 策略选择器
│   │   └── prompts.py             # Prompt 模板

│   │   ├── edu_pptloader.py       # PPT 加载
```
